# QuickRevue (QR) 📱

**QuickRevue** is a high-performance, local-first utility designed to eliminate the hassle of physical QR codes. Whether dealing with faded prints, blurry scans, or a cluttered wallet, QuickRevue allows you to **gather**, **refine**, and **digitize** everything into one crisp interface.

The name embodies our mission: A **Quick** way to **Revue** (organize and display) your essential codes.

---

## ✨ Key Features

- **QR Restoration:** Resolve and reconstruct blurry or damaged QR codes into high-definition digital versions.
- **Consolidated Hub:** Centralize QR codes from multiple sources (paper, screenshots, or manual text) in one place.
- **Manual Entry:** Generate fresh QR codes instantly by inputting raw data or URLs.
- **Local-First:** Privacy by design. All data is stored locally in your browser/device—no cloud, no tracking.
- **Installable Web App (PWA):** _[Future]_ Use QuickRevue like a native app. Access your stored QR codes anytime, even without an internet connection, directly from your home screen.
- **Blazing Fast:** Built with Vite and Bun for a near-instant user experience.

## 🛠️ Tech Stack

- **Framework:** [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Package Manager:** [Bun](https://bun.sh/)

### Local Development (Self-Hosting)

If you prefer to build it yourself, follow these steps:

1. **Prerequisites**
   Ensure you have [Bun](https://bun.sh/) installed on your machine.

2. **Clone the Repository**

    ```bash
    git clone https://github.com/xellanix/quick-revue.git
    cd quick-revue
    ```

3. Install Dependencies

    ```bash
    bun install
    ```

4. Run Development Server

    ```bash
    bun dev
    ```

5. Build for Production

    ```bash
    bun run build
    ```

6. Run Production Server
    ```bash
    bun run preview
    ```
